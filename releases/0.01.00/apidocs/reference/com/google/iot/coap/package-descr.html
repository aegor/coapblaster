CoapBlaster: A experimental CoAP client/server library written in Java.

The most important classes in this package are:

  • Message: A CoAP message
  • Client: A CoAP client
    • Transaction: Tracks an outbound request
  • Server: A CoAP server
    • InboundRequest: Represents an inbound request
    • InboundRequestHandler: Handles inbound requests
    • Resource: Special InboundRequestHandler for folders
    • Observable: Helper for implementing observable resources
