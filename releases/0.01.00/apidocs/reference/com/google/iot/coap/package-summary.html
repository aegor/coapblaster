package

com.google.iot.coap

CoapBlaster: A experimental CoAP client/server library written in Java.

The most important classes in this package are:

  • Message: A CoAP message
  • Client: A CoAP client
    • Transaction: Tracks an outbound request
  • Server: A CoAP server
    • InboundRequest: Represents an inbound request
    • InboundRequestHandler: Handles inbound requests
    • Resource: Special InboundRequestHandler for folders
    • Observable: Helper for implementing observable resources

Interfaces

InboundRequest Object representing an inbound request, providing methods for examining the request and responding to it. 
InboundRequest.Callback Callback interface for receiving additional asynchronous events related to a specific InboundRequest. 
InboundRequestHandler An object which handles inbound requests via a Server instance. 
Interceptor Interface for peeking at the inbound and outbound messages on a LocalEndpoint, possibly choosing to drop them. 
LinkFormat.Provider Complement interface to InboundRequestHandler that allows a parent Resource object to get additional information when building link format listings. 
LocalEndpoint Interface representing a local CoAP endpoint, typically associated with an open port on this machine. 
Observable.Provider Interface that InboundRequestHandler objects can optionally implement in order to communicate their Observable instance to other objects which could use that information, such as Resource objects. 
ProxySelector Interface for determining if and when a proxy should be used on a Client. 
Transaction Interface representing an outbound transaction and keeping track of its pending response.
