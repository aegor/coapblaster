Class Overview

Abstract base class representing a mutable CoAP message. The immutable variant is the base class: Message.

Note that inheriting this class does not make the underlying message mutable: it would be an error to check to see if a message is mutable by doing the check (msg instanceof MutableMessage) because some message implementations start out mutable but become immutable later on.
